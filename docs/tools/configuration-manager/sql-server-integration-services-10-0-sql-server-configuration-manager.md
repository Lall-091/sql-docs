# SQL Server Integration Services 10.0 (SQL Server Configuration Manager)

[!INCLUDE [SQL Server Windows Only](../../includes/applies-to-version/sql-windows-only.md)]

This is the service that supports [!INCLUDE [ssIS](../../includes/ssis-md.md)]. To view the properties of the service, right-click the service in the details pane, and then select **Properties**.

For information about [!INCLUDE [ssIS](../../includes/ssis-md.md)], see [SQL Server Integration Services](../../integration-services/sql-server-integration-services.md).
