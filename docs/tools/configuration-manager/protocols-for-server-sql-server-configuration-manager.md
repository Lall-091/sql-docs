# Protocols for &lt;server&gt; (SQL Server Configuration Manager)

[!INCLUDE [SQL Server Windows Only](../../includes/applies-to-version/sql-windows-only.md)]

To configure the protocols used by the [!INCLUDE [ssDE](../../includes/ssde-md.md)], select **Protocols** for the desired server instance in the console pane, right-click the appropriate protocol in the details pane, and then select **Properties**.
