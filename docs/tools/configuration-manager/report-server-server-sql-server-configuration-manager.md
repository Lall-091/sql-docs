# Report Server &lt;server&gt; (SQL Server Configuration Manager)

[!INCLUDE [SQL Server Windows Only](../../includes/applies-to-version/sql-windows-only.md)]

This is the service that supports [!INCLUDE [ssRSnoversion](../../includes/ssrsnoversion-md.md)]. To view the properties of the service, right-click the service in the details pane, and then click **Properties**.

For information about [!INCLUDE [ssRSnoversion](../../includes/ssrsnoversion-md.md)], see [Reporting Services consolidation FAQ](../../reporting-services/reporting-services-consolidation-faq.md).
