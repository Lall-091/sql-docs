# SQL Server Integration Services Properties (Advanced tab)

[!INCLUDE [SQL Server Windows Only](../../includes/applies-to-version/sql-windows-only.md)]

Microsoft [!INCLUDE [ssISnoversion](../../includes/ssisnoversion-md.md)] is implemented as the [!INCLUDE [ssISnoversion](../../includes/ssisnoversion-md.md)] service. If custom properties are defined, they appear on this tab, with their values. This tab might have no properties listed.
