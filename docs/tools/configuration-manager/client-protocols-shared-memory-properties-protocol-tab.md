# Client Protocols - Shared Memory Properties (Protocol tab)

[!INCLUDE [SQL Server Windows Only](../../includes/applies-to-version/sql-windows-only.md)]

In [!INCLUDE [ssNoVersion](../../includes/ssnoversion-md.md)] Configuration Manager use the **Protocol** tab on the **Shared Memory Properties** dialog box to view or modify shared memory. For more information about connection strings, see [Aliases (SQL Server Configuration Manager)](aliases-sql-server-configuration-manager.md).

## Options

#### Enabled

Possible values are **Yes** and **No**.
