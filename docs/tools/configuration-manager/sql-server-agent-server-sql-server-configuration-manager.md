# SQL Server Agent &lt;server&gt; (SQL Server Configuration Manager)

[!INCLUDE [SQL Server Windows Only](../../includes/applies-to-version/sql-windows-only.md)]

This is the service that supports [!INCLUDE [ssNoVersion](../../includes/ssnoversion-md.md)] Agent. To view the properties of the service, right-click the service in the details pane, and then select **Properties**.

[!INCLUDE [ssNoVersion](../../includes/ssnoversion-md.md)] Agent is a Windows service that executes scheduled administrative tasks called *jobs*. For more information about [!INCLUDE [ssNoVersion](../../includes/ssnoversion-md.md)] Agent, see [SQL Server Agent](/ssms/agent/sql-server-agent).
