# SQL Server Browser Properties (Service tab)

[!INCLUDE [SQL Server Windows Only](../../includes/applies-to-version/sql-windows-only.md)]

The [!INCLUDE [ssnoversion-md](../../includes/ssnoversion-md.md)] Browser program runs as a service on the server.

[!INCLUDE [ssnoversion-md](../../includes/ssnoversion-md.md)] Browser listens for incoming requests for [!INCLUDE [ssnoversion-md](../../includes/ssnoversion-md.md)] resources and provides information about [!INCLUDE [ssnoversion-md](../../includes/ssnoversion-md.md)] instances installed on the computer.

Use the **Service** tab on the **Properties** dialog box to view the following options. All properties except **Start Mode** are read-only.

## Options

[!INCLUDE [service-tab-1](../includes/configuration-manager/service-tab-1.md)]

#### Host Name

Displays the name of the computer or cluster running [!INCLUDE [ssnoversion-md](../../includes/ssnoversion-md.md)] Browser service.
