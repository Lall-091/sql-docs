1. Go to the [Azure SQL hub at aka.ms/azuresqlhub](https://aka.ms/azuresqlhub).
1. Under **SQL Server** select **SQL Server on Azure VMs** to open the **SQL Server on Azure VMs** page.
1. On the **SQL Server on Azure VMs** page, select **+ Create** to open the **SQL Server on Azure Virtual Machines** page.

   :::image type="content" source="media/create-sql-virtual-machine/create-sql-virtual-machine.png" alt-text="Screenshot of the SQL Server on Azure VMs page from the Azure SQL hub page in the Azure portal, showing the +Create button.":::

1. On the **SQL Server on Azure Virtual Machines** page, select an image offer from the drop-down list, and then use **Create virtual machine** to open the **Create a virtual machine** page.
