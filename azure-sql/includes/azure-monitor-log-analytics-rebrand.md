> [!NOTE]  
> This article was recently updated to use the term Azure Monitor logs instead of Log Analytics. Log data is still stored in a Log Analytics workspace and the same Log Analytics service still collects and analyzes the data. The updated terminology better reflects the role of [logs in Azure Monitor](/azure/azure-monitor/logs/data-platform-logs). For more information, see [Azure Monitor terminology changes](/azure/azure-monitor/terminology).
